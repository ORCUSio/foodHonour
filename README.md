# Food Delivery App

A simple food delivery application for Android, built with modern Android development tools. This app showcases a list of food items, allows users to view details, customize their orders, and manage a shopping cart.

## Features

- **Browse Food Items:** The main screen displays a list of available food items in a clear and attractive layout.
- **View Item Details:** Tap on any item to see a detailed view with a larger image, description, price, and nutritional information.
- **Customize Orders:** Users can select different sizes and add extra toppings or ingredients to their orders.
- **Shopping Cart:** A functional shopping cart screen where users can see their selected items, adjust the quantity, or remove items.
- **Order Summary:** The checkout screen provides a detailed summary of the order, including subtotal, taxes, delivery fee, and the final total.
- **Modern UI:** The entire user interface is built with Jetpack Compose, providing a reactive and declarative UI.
- **Seamless Navigation:** Uses Navigation Compose to handle navigation between different screens of the app.

## Technologies Used

- **Kotlin:** The primary programming language for the project.
- **Jetpack Compose:** Android's modern toolkit for building native UI.
- **Material 3:** The latest version of Google's open-source design system.
- **Navigation Compose:** Used for navigating between different composable screens within the app.
- **Coil:** An image loading library for Android to load food images from URLs asynchronously.
- **Gradle:** The build automation tool used for managing dependencies and building the project.

## How to Run

1.  Clone this repository.
2.  Open the project in a recent version of Android Studio.
3.  Let Gradle sync and download the required dependencies.
4.  Build and run the app on an Android emulator or a physical device.

*Note: An internet connection is required to load the food images.*
